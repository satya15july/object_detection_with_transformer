# Introduction
This project try to use Detectron2 and Huggingface framework to do object detection using transformer based architecture.

With Huggingface framework, the following transformer based architectures are used

-   DETR(Detection Transformer).
-   Conditional DETR
-   Deformable DETR.
-   YOLOS.

Please refer "__object_detection_with_hugginggface/README.md__" for more information.


With Detectron2 framework, DETR and DynamicHead architectures are used.Please refer "__object_detection_with_hugginggface/README.md__" for more information.
